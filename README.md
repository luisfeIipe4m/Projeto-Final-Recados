\# Lista de Recados
Aplicação fullstack com React no front-end e Laravel no back-end.

\## Integrantes

\- Luis Felipe Barbosa da Silva

\- Dimitri Barbera Guimarães

\- Matheus Cappi

## Tecnologias
- React
- Laravel
- MySQL
- Sanctum (autenticação via token)

## Funcionalidades
- Cadastro e login de usuários
- Criação, listagem e exclusão de recados
- Cada usuário só vê os próprios recados

## Como rodar o projeto

### Frontend
\`\`\`
cd frontend
npm install
npm start
\`\`\`

### Backend
\`\`\`
cd backend
composer install
php artisan serve
\`\`\`
